# install-run-xray

Скрипт для **установки и запуска Xray-сервера** (VLESS + Reality) в ознакомительных и учебных целях.  
При запуске автоматически устанавливает Xray, генерирует ключи и UUID, создаёт конфиг и запускает службу.

---

### 🚀 Установка и запуск

```bash
wget https://raw.githubusercontent.com/akaC1J/install-run-xray/main/install-run-xray.sh && chmod +x install-run-xray.sh && sudo ./install-run-xray.sh
````

---

### 📋 Prerequisites

* Требуются права `sudo`
* Установленная утилита `wget`
* Протестировано на:

  * Ubuntu 22.04
  * Ubuntu 24.04
  * Debian 12
  * Debian 13

---

> ⚠️ Скрипт предназначен для тестовых и учебных целей, для ознакомления с протоколом VLESS
